Descrição detalhada

template<typename T>
class Lista< T >

Classe que representa uma Lista Encadeada.

Essa classe implementa uma Lista Encadeada, utilizando dos nós implementados na classe No para armazenar uma lista de elementos. Essa estrutura de dados foi escolhida para estimular o aprendizado de ponteiros e alocação dinâmica na linguagem C++.
